Sección 1: Cómo empezar

Con está aplicación puedes hacer 5 tipos de acciones


  • En el primer apartado se encuentra la opción de introducir una URL ,
    Si diesemos al botón de EJECUTAR, realizará los comandos que la aplicación realiza que son:
    • - PING
    • - CURL
    • - TRACERT
    • - NSLOOKUP

  • Una vez terminado la ejecución de todos los comandos en las pestañas que se encuentran debajo volcará todos los datos que deberian de salir tras realizarse dichos comandos


  • Para mostrar la página introducida en el explorador de internet , con dar al botón Mostrar página seria suficiente


Sección 2: Funcionalidades Principales

La página dispone de atajos para realizar todas las tareas que son necesarias , os dejo un apartado donde podeis ver como están asignados todos los atajos y donde se dirigen

  • Al Realizar:

    - Ping , es el comando TCP/IP principal que se usa para solucionar problemas de conectividad, disponibilidad y resolución de nombres

    Es una herramienta de diagnóstico que permite hacer una verificación del estado de una determinada conexión o host local.

  • - Curl , permite al usuario realizar solicitudes a un servidor y recuperar datos desde la línea de comandos. Al usar CURL, cualquier usuario puede, entre otras acciones:

    • Obtener el contenido de una página web.

      • En nuestro caso estas 4 opciones las podriamos realizar en las siguientes actualizaciones

    • Subir o descargar archivos.
    • Enviar datos a un formulario.
    • Autenticarse en servicios web.
    • Realizar comprobaciones de estado de páginas web.

  • - TRACERT , es una utilidad de línea de comandos que se usa para determinar la ruta que toma un paquete de protocolo de Internet (IP) para alcanzar su destino

  • - NSLOOKUP , es una herramienta de línea de comandos muy práctica y fácil de usar, cuya función básica es encontrar la dirección IP de un equipo determinado o realizar una búsqueda DNS inversa

  • En la barra de explorador existe la opción de properties aqui se pueden modificar los distintos tipos de saltos del comando Tracert , el tiempo en milisegundos y el tiempo en milisegundos de cada salto del Ping



Sección 3: ¿Cómo Guardar los comandos?

Para guardar todos tus comando tendrás que darle a la opción de File , De ahí tienes 2 opciones de guardado , existe la posibilidad de guardar cada comando en un fichero cada uno o bien dando a Save as puedes guardar todo en un solo archivo o bien en Save que te dará la opción de guardar los distintos comandos , ping / curl / tracert / nslookup. cada vez que quieras guardar algo tendrás que elegir una ruta de guardado. te dejo unas imagenes para sea mas facil y intuitivo de ejecutar
